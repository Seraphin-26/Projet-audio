🎧 Aura Sound X – Premium Product Launch

Une landing page immersive de lancement de produit "Luxe & Tech", conçue avec Next.js 15, Tailwind CSS et Framer Motion. Ce projet met l'accent sur les animations fluides, l'expérience utilisateur (UX) et la performance.
✨ Caractéristiques principales

    Design Futuriste & Minimaliste : Utilisation d'une esthétique haut de gamme inspirée des plus grandes marques de tech.

    Animations de Scroll (Parallax) : Intégration de Framer Motion pour des transitions fluides et un effet de profondeur au défilement.

    Bento Grid Layout : Présentation des caractéristiques techniques via une grille moderne et responsive.

    Thème Dynamique : Support complet du mode Sombre (Dark) et Clair (Light) avec des transitions douces.

    Optimisation SEO & Performance : Structure sémantique HTML5 et optimisation Next.js.

🛠️ Stack Technique

    Framework : Next.js (App Router)

    Styling : Tailwind CSS

    Animations : Framer Motion

    Icônes : Lucide-React

    Thèmes : Next-Themes
